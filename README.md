## Install from Source

- Clone/Download this repo.
- Extract the zip file if you didn't go clone route.
- Go to `chrome://extensions/`.
- Enable `Developer mode` switch.
- Click on `Load unpacked`.
- Select the extracted-folder.
- Pin it if you want.
- Done.
